# TicTacToe

Creating a personal profile where statistics are displayed and you can view all games by clicking on any of the profiles in the profile.
The game is up to 3 points.

## Front

- ReactJs
- Redux
- MaterialUi
- Socket.io-client
- JWToken

## Back

- NodeJs
- Express
- Mongoose
- JWToken
- Socket.io
